Step 1: Create Database

Open XAMPP and start Apache and MySQL.

Open phpMyAdmin in your browser.

Click “New” on the left side.

Type the database name: nobuko

Click “Create”.


Step 2: Place the Project Folder

Copy your whole project folder.

Go to this location on your computer: C:\xampp\htdocs

Paste the project folder there.
Example: C:\xampp\htdocs\nobuko



Step 3: Run the Project

Open your browser.

Type this in the address bar: localhost/nobuko

Press Enter.

The project will start running.


Step 4: (Optional) Import SQL File
If you have a database file named nobuko.sql,

Open phpMyAdmin.

Select the nobuko database.

Click on the “Import” tab.

Choose the file nobuko.sql.

Click “Go”.
